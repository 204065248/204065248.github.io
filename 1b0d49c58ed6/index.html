前置工作

本脚本支持同步自己的仓库和starred仓库，使用前请确保你的电脑已经安装了git，并且能够正常的拉取自己仓库中的代码。

Token获取方式

  1. 跳转到 https://github.com/settings/tokens
  2. 点击 Generate new token
  3. 建议给全部权限，不然可能会出现同步不全的情况
  4. 令牌失效时间拉满，防止经常维护
