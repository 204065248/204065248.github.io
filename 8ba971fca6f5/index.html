vcpkg安装指定版本教程

发表于2023-09-15|更新于2023-12-29|C/C++小技巧
|阅读量:

1、vcpkg\versions中查看所需要的版本号

2、在vcpkg.exe目录下创建vcpkg.json文件

3、编写文件

4、在powershell中设置编译环境，这里示例为64位静态库版本 $env:VCPKG_DEFAULT_TRIPLET = 'x64-windows-static'

5、在vcpkg根目录下使用vcpkg install

6、完成后删除vcpkg.json文件
